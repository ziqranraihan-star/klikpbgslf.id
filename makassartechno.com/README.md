# Makassar Techno

Agency website built with Next.js 14 (App Router), TypeScript, Tailwind CSS, and lucide-react.

## Getting Started

```bash
npm install
npm run dev
```

## Structure

- `app/` - App Router pages and layout
- `components/` - UI sections and shared components
- `data/site.ts` - Centralized content data
- `public/` - Logo, favicon, and portfolio placeholders

## Notes

All CTA WhatsApp links are generated from `data/site.ts` and include service + package context.
